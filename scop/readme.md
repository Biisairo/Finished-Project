# obj compile

## Build
```
make .
```

## run
```
./obj_compile <path to file in ./map>
```

## Key
front - ```arrow front```

back - ```arrow back```

left - ```arrow left```

right - ```arrow right```

up - ```mouse left click```

down - ```mouse right click```

view - ```mouse move```

zoom in/out - ```mouse wheel```

zoom reset - ```mouse wheel click```

color <-> texture - ```M```

change texture - ```SPACE```

draw face <-> draw line - ```,```
